Documnation of toys

Method Route Descrption Links
GET /toys Returns a list of toys according to the query parameters.
  • page (optional): page number to return (default: 1).

    http://localhost:3003/toys?page=1

  • perPage (optional): number of toys per page (default: 4, maximum: 10).

    http://localhost:3003/toys?perPage=4

  • sort (optional): field to sort by (default:id).

    http://localhost:3003/toys?sort=id

  • reverse (optional): if set to yes, sorts in descending order.

    http://localhost:3003/toys?sort=price&reverse=yes

  • s the search query to match against toy names or descriptions.

    http://localhost:3003/toys?s=Teddy
GET /toys/cat Returns a list of toys by category.
  • cat the name of the category to retrieve toys for.

    http://localhost:3003/toys?cat=car
GET /toys/price Returns a list of toys filtered by price range.
  • max (optional): minimum price to filter by (default: 0).
  • min (optional): maximum price to filter by (default: maximum 999999).

    http://localhost:3003/toys/price?min=100&max=1000
GET /toys/count Returns how many items there are
  • count the amount of items

    http://localhost:3003/toys/count
GET /toys/single/:id Returns a single toy by its ID.
  • id the ID of the toy to retrieve.

    http://localhost:3003/toys/single/64590e6d78b4c165f872d618
POST /toys Adds a new toy. Token is required!
  • Request Body Toy data to be added in the following format.
    • {
  • Name: Name.. (String)
  • Info: Info.. (String)
  • Category: Category.. (String)
  • Img_url: is allowed and not required. (String)
  • Price: 1-999999 (Number)
  • User_id: id of the user who wants to add an item. (String)
    • }
PUT /toys/:id Updates an existing toy. Token is required!
  • id the ID of the toy to update.
  • Request Body Toy data to be updated in the following format.
    • {
  • Name: (String)
  • Info: (String)
  • Category: (String)
  • Img_url: is allowed and not required. (String)
  • Price: 1-999999 (Number)
    • }
    (Only an administrator and the person who added the product will be able to change details.)
DELETE /toys/:id Deletes a toy by its ID. Token is required!
  • id the ID of the toy to delete.

    • (Only an administrator and the person who added the product will be able to change details.)
